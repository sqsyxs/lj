第二十六章 锅子都蕴含大道之理
　　提到吃饭，众人这才从对联的震惊中回过神来。
　　赵山河和白无尘互相对视一眼，抿了抿嘴吧，思绪复杂。
　　这高人实在是太高了，仙人恐怕已经不足以形容他了吧。
　　不说其他的，就算只是从他这里求一个字，那都是无上机缘，可以当成传家之宝，造福后辈。
　　赵山河心痒难耐，低声道：“白无尘，你不是的从李公子那里得了一幅画吗？赶紧拿出来给我瞅瞅。”
　　“现在怕是不行了。”白无尘苦着脸，眼中都溢出了泪水。
　　“咋地？我们也这么多年交情了，你还想对我私藏？看一眼都不给？”赵山河气愤道。
　　“不是不给，那副画……被剑魔给毁了。”白无尘苦涩道。
　　“什么？！”
　　赵山河的老脸顿时就红了，胡子竖朝天，指着白无尘的鼻子破口大骂，“你个渣渣，这等至宝你都保护不了，那副画没了，你怎么还活着，好意思吗你！”
　　白无尘的内心在滴血，这是他这辈子最不愿意想起的事情，“赵老头，你以为我想啊，我当时恨不得死的是我，也不愿意画出事啊！”
　　他年纪也几百岁了，此时双目含泪，随时都会哭出来。
　　那副画足以让万剑仙宗万世不朽啊！
　　赵山河知道白无尘是真的伤心，换做他自己，也肯定不好过，最终只能发出一声长长的叹息，“哎！”
　　“你们两个长吁短叹的做什么呢？吃饭一定要保持好的心情，不然是对美食的侮辱。”李念凡笑着看着他们二人。
　　说者无心，听者有意。
　　“李公子……”
　　白无尘看向李念凡，眼睛红红，目光中居然涌出了感动的泪水。
　　李公子这是在提点自己啊！
